## Installation:
* Python Language:

	* Manually:
		```bash
		sudo apt-get install python3 python3-venv python3-pip -y
		```

	* Using ShellScript:
		```bash
		cd Python
		chmod +x install.sh
		sudo ./install.sh
		```

## Run Python code:
```bash
python3 ./{ProgramName}.py
# Example: python3 ./main.py 
```
